# mc-craftbot
A simple chatbot that helps you craft items in Minecraft step by step

## Project Structure

```
mc-craftbot/
├── src/
│   ├── bot/        # Core bot entry point and lifecycle
│   ├── commands/   # Chat command handlers
│   ├── crafting/   # Crafting recipe logic
│   └── utils/      # Shared utility functions
├── config/         # Configuration files
├── tests/          # Automated tests
├── docs/           # Project documentation
├── .gitignore
├── package.json
└── README.md
```

## Getting Started

```bash
npm install
npm start
```

## Running Tests

```bash
npm test
```
